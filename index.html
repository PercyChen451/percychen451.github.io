Project Overview

Designed and implemented a closed-loop DC-DC converter using discrete analog components to reverse-engineer commercial IC architecture. Constructed a 5-op-amp PID controller to actively regulate output voltage, balancing transient response speed with stability against noise.

Engineering Highlights

  • Analog Control Loop: Built a continuous-time PID controller with independent P, I, and D stages to eliminate steady-state error and reduce overshoot.
  • PWM Generation: Engineered a custom duty-cycle generator using a comparator and BJT-amplified ramp signal.
  • System Performance: Achieved <3% voltage tracking error in the 1.3V‚Äì3.0V active range with low output ripple (~20mV).
